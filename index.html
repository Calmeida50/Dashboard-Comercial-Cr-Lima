Importar Relatórios

Arraste os relatórios das indústrias para atualizar o dashboard automaticamente

📁

Arraste os arquivos aqui

Relatórios Excel (.xlsx) ou PDF das indústrias
Granado, Prudence, Belliz, Cless, Kisabor, Ever Green, Fiat Lux...


ℹ️ Como Funciona a Importação

1. Baixe o relatório do portal da indústria (Excel ou PDF)

2. Arraste o arquivo para a área acima ou clique em "Selecionar Arquivos"

3. O sistema identifica automaticamente a empresa e o período pelo nome do arquivo

4. Os dados são processados e o dashboard é atualizado instantaneamente


Formatos suportados: XLSX, XLS, CSV, PDF

Empresas reconhecidas: Granado, Prudence, Belliz, Cless, Kisabor, Ever Green, Fiat Lux, Aquafast, Payot, Depimiel, Botânica


⚠️ Os dados importados são armazenados localmente no seu navegador. Para compartilhar com a equipe, exporte o JSON e atualize o repositório GitHub.
